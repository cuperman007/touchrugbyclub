Getting started

How to create your own website, using a Touch Rugby Club Network template.

Technical knowledge

You will need someone that knows how to create a website using HTML, CSS and JavScript and how push changes to a remote repository using GIT.

Erm, what?

Don't know what any of this means?
Stop here and contact us

Follow these steps

  • Using Git, fork an existing Touch Rugby Club Network website. For example: Staffordshire Stormers source
  • Push the your forked code to your own GitHub repository
  • Configure your repository to use GitHub pages
  • Contact us to let us know what subdomain you want to use, for example; myclubname.touchrugby.club. We will configure the DNS for you.
  • Make changes to your local copy of code
    • css/Site.css - all the main colours are controlled via CSS variables
    • Index.html - all the contents are in one page
    • Images/ - folder containing any images you are using
    • Scripts/ - folder containing any scripts you are using
    Note: if you are using our template, please do not remove the Touch Rugby Club Network nor LiquidJelly links and references. These should remain without change from the original source.
  • Push your changes
  • You are done !
